#Frontend   

A equipe de UX desenvolveu componentes padrões que são utilizadas para o desenvolvimento de telas customizadas. 
Você encontra os componentes no [Style Guide do Fusion](https://www.neomind.com.br/fusion/styleguide/#/styleguide/dashboard).

## Frameworks

Quem determina os padrões e tecnologias utilizadas no produto é a equipe de inovação. Atualmente os frameworks homologação são:

- AngularJS [documentação AngularJs](https://docs.angularjs.org/api).
- Bootstrap [documentação Bootstrap](https://getbootstrap.com/docs/3.3/).
- UI Bootstrap inclui [componentes Bootstrap integrados com Angular](https://angular-ui.github.io/bootstrap/).

Mais informações de padronização da plataforma, você encontra no Pages da equipe de inovação, acessando [aqui](/docs/).
