# Exercício

Para fixar o uso dos Converters, no formulário da solicitação de viagem:

-   Utilizar um campo de CEP
-   Ao preencher o CEP, devem ser preenchidos outros campos de forma automática.

Campos:

-   Rua
-   Cidade
-   Estado

Para obter as informações necessárias com o CEP, utilize uma API de CEP ([sugestão](https://postmon.com.br/))

Para realizar a requisição, é possível utilizar os métodos _Fetch_ ou _callSync_ do JavaScript puro, ou _Ajax_ do JQuery.

[Documentação Ajax](https://api.jquery.com/category/ajax/)

[Documentação Fetch](https://developer.mozilla.org/en-US/docs/Web/API/Fetch_API/Using_Fetch)
