# NeoObject

No Fusion, sempre que trazemos algum campo de um formulário para uma classe java, esse campo deve ser armazenado em um objeto de um tipo específico, por exemplo, datas são objetos GregorianCalendar, números decimais são objetos BigDecimal. Os formulários são objetos NeoObject, então quando referênciamos um campo do tipo E-Form em nossa classe Java, tratamos como um objeto NeoObject.

- NeoObjects são campos do tipo formulário dinâmico
- NeoObjects são genéricos, assim como objetos genéricos Java
