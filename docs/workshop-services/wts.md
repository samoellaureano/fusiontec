# Workshop Técnico de Serviços
O WTS foi criado com o intuito de compartilhar conhecimento através de vídeos técnicos criados pela equipe de Serviços sobre customizações no Fusion Platform desenvolvido pela Neomind.

## I WTS - GRID EForm List

Acesse o vídeo [aqui](https://youtu.be/bO8bz5f7J4k)

Acesse os códigos utilizados [aqui](https://gitlab.neomind.com.br/neomind-servicos/utils/WTS/i-wts-eform-list-grid/i-wts)

Os tópicos abordados foram:

- Configurações do Campo Lista (selecionável, crável/Removivel, Formulas)
- Customizar Botões de uma grid (Incluir novos botões, exibir/ocultar os botões)
- Customizar Itens removíveis da Grid
- Customizar Itens editáveis da Grid
- Customizações Gerais
