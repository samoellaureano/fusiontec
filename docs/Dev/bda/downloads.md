
## 1. Fazendo download do Microsoft SQL Server

Para fazer o download do microsoft sql server, basta acessar o [link](https://www.microsoft.com/en-us/sql-server/sql-server-downloads) e procurar por **express**.

![img](IMG/download/express-sql-server-download.png)

Após o download comece a instalação, apenas dando próximo até o fim.

## 2. Fazendo download o SMSS

Para fazer o download do SMSS basta acessar este [link](https://learn.microsoft.com/en-us/sql/ssms/download-sql-server-management-studio-ssms?view=sql-server-ver16) e em seguida clicar em **Free download**

![img](IMG/download/downloads%20smss.png)

Após download, basta seguir a instalação normalmente, sem segredo.
